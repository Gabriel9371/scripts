# Scripts

Este repositório reune **scripts úteis** que desenvolvi para estudo e uso pessoal em ambientes Linux.

A ideia aqui **não é criar projetos grandes**, frameworks ou bibliotecas,
mas sim pequenos scripts que resolvem problemas reais do dia a dia
enquanto eu aprendo e aprofundo meus conhecimentos.

Alguns desses scripts podem ser úteis para outras pessoas,
então resolvi centralizar tudo em um único repositório.

---

## Organização

Os scripts são organizados por linguagem.
Cada script fica em seu próprio diretório, com um README explicando
o que ele faz e como usar.

```text
scripts/
├── lua/        # Scripts feitos com Lua
├── python/     # Scripts feitos com Python (futuro)

```
## Observação
- os scripts foram feitos pra estudo e uso pessoal
- melhorias e ajustes vão ser feitos com o tempo :)
